conda、Anaconda、miniconda、Bioconda的区别

Conda is a portable package manager primarily for python and precompiled binaries. Miniconda is the base system of conda. It includes a standard python and a few required dependencies such as readline and sqlite. In conda, a channel contains a set of software typically managed by the same group.

  1. conda is a package manager,
  2. Miniconda is the conda installer
  3. Anaconda is a scientific Python distribution that also includes conda
  4. Biconda is a channel of conda focusing on bioinformatics software. Bioconda主页
