4.3 Monocle相关函数功能

preprocess_cds

first normalizes the data by log and size factor to address depth differences, or by size factor only. Next, calculates a lower dimensional space that will be used as the input for further dimensionality reduction like tSNE and UMAP. 首先用log归一化数据，然后计算PCA, num_dim参数取PCA的维度，计算PCA的目的是为了以后在PCA基础上继续 tSNE或者UMAP降维。

align_cds

重要的参数是alignment_group， residual_model_formula_str， 其中alignment_group是去除批次效应， 而residual_model_formula_str一般是背景噪声，这里是用lmFit 函数 对表达数据 关于背景参数的回归，然后表达数据减去这个数据。具体原理为什么这样，还是不清楚。

cluster_cells

对cells做划分， 得到的结果是”cluster”和“partition“。 其中”cluster” 部分是无监督聚类（Louvain/Leiden communities）结果，然后针对”cluster” 得到 “partition“（一个聚合的更大的类）represent superclusters of the Louvain/Leiden communities that are found using a kNN pruning method。通过 clusters(), partitions() 函数可以取到对应的分类结果.结果也可以通过 cds$cluster

learn_graph

学习进化轨迹函数

aggregate_gene_expression

聚合表达 在这里是求 每个基因模块 每个 分组中的平均值。
